# 配色方案

## 主色调
- primary: #2C3E50（深灰蓝，用于主文本、标题、轴线）
- secondary: #C0392B（红，用于强调、预警、高铁显著便宜段）
- success: #16A085（绿，用于正向指标、高铁二等座线）
- info: #2980B9（蓝，用于RPA线、链接、信息提示）
- warning: #F39C12（橙，用于节假日阴影、警告状态）
- purple: #8E44AD（紫，用于辅助区分）

## 中性色
- bgPrimary: #FFFFFF（主背景）
- bgSecondary: #F2F3F4（次级背景、卡片背景）
- border: #B2BABB（边框、分割线）
- textPrimary: #2C3E50（主文本）
- textSecondary: #566573（次级文本）
- textMuted: #95A5A6（弱化文本）

## 图表专用
- chartLine1: #2C3E50（航空均价线）
- chartLine2: #2980B9（RPA线）
- chartLine3: #16A085（高铁二等座）
- chartLine4: #C0392B（高铁商务座）
- chartLine5: #8E44AD（高铁一等座）
- weekendShade: rgba(168, 216, 234, 0.25)（周末阴影）
- holidayShade: rgba(255, 211, 182, 0.35)（节假日阴影）
- kdeLine: #C0392B（核密度曲线）
- medianLine: #2980B9（中位数线）
- meanLine: #2C3E50（均值线）

## 热力图渐变
- heatmapMin: #2980B9（蓝，低价）
- heatmapMid: #F2F3F4（白，中价）
- heatmapMax: #C0392B（红，高价）

## SHAP配色
- shapHigh: #C0392B（特征值高）
- shapLow: #2980B9（特征值低）
- shapNeutral: #B2BABB（中性）

## 状态色
- statusUp: #C0392B（上调）
- statusDown: #16A085（下调）
- statusHold: #F39C12（维持）
